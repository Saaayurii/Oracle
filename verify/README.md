# Проверка лабораторных работ (Node.js)

Скрипт автоматически проверяет выполнение Лаб. 2, 3 и 4 в Oracle XE.

## Требования

- Node.js ≥ 18
- Запущенный Oracle XE контейнер (`docker compose up -d` из корня репо)
- Oracle Instant Client **не нужен** — используется thin-режим oracledb v6

## Установка

```bash
cd verify
cp .env.example .env
npm install
```

## Запуск

```bash
# Все лаборатории
npm run verify

# Только лабораторная 2
npm run verify:lab2

# Только лабораторная 3
npm run verify:lab3
```

## Что проверяется

### Лаб. 2 — Таблицы и данные
- Существование всех 6 таблиц
- Минимальное количество строк в каждой таблице
- Целостность внешних ключей
- Ограничения CHECK (price > 0)

### Лаб. 3 — Процедуры и функции
- Наличие всех 4 процедур и 5 функций (статус VALID)
- Вызов `get_customer_total` → результат ≥ 0
- Вызов `get_book_availability` → непустая строка
- Вызов `calculate_discount` с 4 граничными значениями
- Вызов `get_author_full_name` → непустая строка
- Вызов `get_top_book` → непустая строка
- Выполнение `place_order` → получение order_id
- Выполнение `cancel_order` → отмена тестового заказа
- Выполнение `update_stock` → изменение остатка с проверкой

### Лаб. 4 — APEX-запросы
- Работоспособность всех SQL-запросов, используемых на страницах APEX
